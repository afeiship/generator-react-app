# generator-react-app
> Yeoman generator for react app.

## installation
```bash
npm install -g yeoman
git clone git@github.com:afeiship/generator-react-app.git
cd generator-react-app
npm link
```

## usage
+ component/module/module-multi/service:
```bash
yo react-app:view
yo react-app:service
yo react-app:mixin
```
